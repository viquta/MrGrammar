# Security Policy

## Supported Versions

This project is in public beta. Only the latest version on the default branch is actively maintained.

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report issues privately by opening a [GitHub Security Advisory](../../security/advisories/new) in this repository. Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept (if safe to share)
- Any suggested remediation

You can expect an acknowledgement within a few days. Given the beta status of the project, response and fix timelines may vary, but all reports will be taken seriously.
