# Contributing to MrGrammar

Thanks for your interest! This project is in public beta and still under active development, so contributions and feedback are welcome.

## Ways to contribute

- **Bug reports** — open a GitHub issue with steps to reproduce and what you expected vs what happened.
- **Feature suggestions** — open an issue describing the idea and the problem it solves.
- **Pull requests** — for anything beyond a small fix, please open an issue first so we can discuss the approach.

## Local setup

See the [Quick Start](README.md#quick-start) section in the README. Docker Compose is the easiest path.

## Code style

- Python: follow PEP 8; the project uses Django conventions throughout.
- TypeScript/Svelte: follow the existing patterns in `frontend/src/`.
- Keep commits focused and write a short, descriptive commit message.

See [SECURITY.md](SECURITY.md).
